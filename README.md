# טופס יצירת קשר (Contact Form)

## תיאור
פרויקט זה מכיל טופס יצירת קשר בעברית עם תמיכה מלאה ב-RTL ושליחת מיילים אוטומטית.

## קבצים
- `index.html` - דף HTML עם טופס יצירת קשר
- `Code.gs` - Google Apps Script לטיפול בשליחת המיילים

## התקנה והגדרה

### שלב 1: הגדרת Google Apps Script
1. היכנס ל-[Google Apps Script](https://script.google.com/)
2. צור פרויקט חדש
3. העתק את התוכן מקובץ `Code.gs` לעורך הסקריפט
4. עדכן את ההגדרות בתחילת הקובץ:
   - `OWNER_EMAIL` - כתובת האימייל של בעל האתר (לקבלת הפניות)
   - `SITE_NAME` - שם האתר שלך

### שלב 2: פרסום ה-Web App
1. לחץ על "Deploy" > "New deployment"
2. בחר "Web app" כסוג הפריסה
3. הגדר:
   - Execute as: Me
   - Who has access: Anyone
4. לחץ "Deploy" והעתק את ה-URL שקיבלת

### שלב 3: עדכון ה-HTML
1. פתח את קובץ `index.html`
2. מצא את השורה עם `<form id="contactForm" action="..."`
3. החלף את ה-URL בשדה `action` ב-URL שקיבלת מהשלב הקודם

### שלב 4: העלאת הדף
העלה את קובץ `index.html` לשרת האחסון שלך או השתמש ב-GitHub Pages.

## איך זה עובד

1. **משתמש ממלא את הטופס** - שם, אימייל והודעה
2. **שליחה לשרת** - הטופס נשלח ל-Google Apps Script
3. **שליחת מיילים**:
   - מייל לבעל האתר עם פרטי הפנייה
   - מייל אוטומטי לשולח עם אישור קבלה
4. **תגובה למשתמש** - הצגת הודעת הצלחה או שגיאה

## תכונות

- ✅ עיצוב מודרני ומותאם למובייל
- ✅ תמיכה מלאה ב-RTL (עברית)
- ✅ אימות נתונים בצד לקוח
- ✅ שליחת מיילים HTML מעוצבים
- ✅ מייל אוטומטי לשולח
- ✅ הודעות משוב למשתמש
- ✅ ללא צורך בשרת backend משלך

## דרישות

- חשבון Google (עבור Google Apps Script)
- שרת אחסון סטטי (או GitHub Pages)
